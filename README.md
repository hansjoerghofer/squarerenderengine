# Square  Render Engine
3D real-time render engine

Implemented Features:
* Physically Based Rendering (PBR)
* Image Based Lighting (IBL)
* Directional Shadow Mapping 
  * Soft Shadows (PCF & PCSS)
* Post Processing
  * Bloom 
  * Tonemapping
